## 💎 Why AuditLens?

Traditional compliance audits are manual, fragmented, and slow. **AuditLens** transforms your raw evidence directory into a production-grade, interactive audit workspace using agentic AI. 

It's not just a script; it's a "lens" into your audit readiness—built with an obsession for **veracity**, **speed**, and **impeccable design**.

## 🚀 Key Features

- **🧠 Agentic Intelligence**: Classifies documentation, maps controls, and tracks provenance automatically.
- **🎨 Impeccable UI**: A premium, editorial-grade interactive workspace powered by OKLCH color spaces and fluid responsive design.
- **🛡️ Compliance Crosswalks**: Native support for ISO 27001, SOC 2, HIPAA, and NIST CSF.
- **🔌 Multi-Agent Ready**: Direct integration for Antigravity, Gemini, Codex, and ChatGPT via an included MCP server.
- **🔍 Truth-First Reporting**: Eliminates "AI slop" in documentation. Generates punchy, technical, and accurate findings.

## 🎨 Interactive Workspace

AuditLens generates a self-contained, portable HTML workspace that represents your entire audit posture. No generic card grids—just pure, editorial visual hierarchy.

## 📦 Installation

Get running in seconds with the Claude Code plugin or directly via CLI.

### Option 1: Claude Code (Recommended)
```bash
/plugin marketplace add adhit-r/audit-lens
```

### Option 2: Manual Setup
```bash
git clone https://github.com/adhit-r/audit-lens.git
cd audit-lens
# Follow the INSTALL.md for your specific agent
```
